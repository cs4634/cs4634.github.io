# 沈辰熹个人作品集

纯静态个人网站，可直接免费部署至 GitHub Pages、Cloudflare Pages 或 Netlify。

## 本地预览

直接双击 `index.html`，或在本目录运行任意静态文件服务器。

## GitHub Pages

1. 新建 GitHub 仓库并上传本目录内容。
2. 打开仓库 `Settings → Pages`。
3. 在 `Build and deployment` 中选择 `Deploy from a branch`，分支选择 `main` 和根目录。
4. 保存后等待 GitHub 生成公开网址。

公开前请检查邮箱、手机号和简历文件是否适合公开展示。
